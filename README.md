# Owlette

WFP Location Finder javascript client.


## Simple usage

Example:

    <script src="owlette.min.js"></script>
    <script>
        var owl = new Owlette();

        owl.search('Rome')
            .done(function(results){})
            .fail(function(error){})
    </script>
